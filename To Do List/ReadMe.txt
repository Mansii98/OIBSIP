To-Do List Application
This repository contains the source code for the To-Do List application developed by Oasis Infobyte. The application is designed to help users keep track of their tasks and manage their daily activities efficiently.

Features
Task Management: Users can create, update, and delete tasks.
Task Prioritization: Tasks can be assigned priority levels to indicate their importance.
Due Dates: Each task can have a due date associated with it.
Categories: Users can categorize tasks into different categories for better organization.
Reminders: Users can set reminders for their tasks to receive notifications.
Search and Filter: The application provides search and filter functionality to easily find specific tasks.
User Accounts: Users can create accounts to store their tasks and access them from multiple devices.
User-Friendly Interface: The application has an intuitive and user-friendly interface for a seamless task management experience.
